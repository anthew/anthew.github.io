Biography

As a Software Engineering major, I’ve spent significant time studying object-oriented programming principles and popular design patterns. So far, I’ve used Java, C++, and Python. I’ve also done some web application development using JavaScript, HTML, and CSS. In addition, I have studied database design and queries using SQL.
        


        
Right now, I’m working on my capstone project with a team to create a website called SongCanvas for users to create music videos. This is a 2-semester Agile development experience consisting of requirements elicitation, software construction, and deployment stages. We are using Express.js to host a Node.js server that runs on a Google App Engine instance. This server connects to a Google Cloud SQL instance to query a MySQL relational database. In this group project, I’ve been taking the initiative in communicating my comments, questions, and feedback to my teammates. I strive for an environment where everybody in the team has a say. I want to be the leader that encourages each team member to voice their ideas and feedback to the group.
